#####4.确定申请状态码的定义



| 状态码 | 状态   | 说明                                 |
| ------ | ------ | ------------------------------------ |
| 0      | 已提交 | 法院本地提交案件                     |
| 1      | 已发送 | 法院推送案件信息给仲裁院成功         |
| 2      | 已接收 | 法院收到仲裁院推送的案件信息         |
| 3      | 已结案 | 法院案件结案                         |
| 4      | 已完成 | 法院案件信息推送后，确认仲裁员以接收 |
| 5      | 已反馈 | 法院在3个工作日收到仲裁院的审核确认  |
